# Problem 1: Water Bill Generator

## Objective

Create a program that calculates the water bill based on water consumption.

---

## Billing Rules

| Consumption Range | Rate Per Unit |
|------------------|---------------|
| First 50 Units | Rs 2 per unit |
| Next 50 Units | Rs 3 per unit |
| Above 100 Units | Rs 5 per unit |

---

## Requirements

Write a program that:

1. Accepts water consumption units from the user.
2. Calculates the bill according to the given slab rates.
3. Displays the final water bill.

---

## Sample Input

```text
120
```

## Sample Output

```text
Water Bill = Rs 310
```

---
